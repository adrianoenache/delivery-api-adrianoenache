# Arquitetura de Sistemas Java - Projeto delivery API

Repositório dos exercícios. [Link para material do curso e referências](https://github.com/adrianoenache/Arquitetura-de-Sistemas-Java).

## Comandos

Inicie o servidor com

```bash
mvn spring-boot:run
```
